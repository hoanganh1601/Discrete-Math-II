# Discrete Math II - CodePTIT Solutions

This repository contains my C++ solutions for the **Discrete Mathematics II** exercises on the CodePTIT platform. 

The primary focus of this module is **Graph Theory** and its algorithmic applications.

## 📚 Topics Covered

The solutions in this repository are categorized based on the core graph theory concepts covered in the course:

*   **Graph Representation & Basics** 
*   **Graph Traversal & Connectivity** 
*   **Articulation Points & Bridges** 
*   **Eulerian & Hamiltonian Cycles** 
*   **Spanning Trees** 
*   **Shortest Path Algorithms** 

## 🗂️ Structure

Each file in this repository corresponds to a specific problem on CodePTIT. 
*   **File Naming Convention:** `[Problem_Code].cpp` (e.g., `TRR1001.cpp`)

## 🛠️ Technologies
*   **Language:** C++
*   **Concepts:** Data Structures, Graph Theory, Algorithms
